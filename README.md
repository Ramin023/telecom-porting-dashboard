# Telecom Automation Tools

This repository contains automation tools for telecom number porting workflows.

## Features

- Python script to simulate pulling telephone number status from carriers
- Excel VBA automation for processing telecom order data
- Sample dataset for testing

## Technologies

- Python
- Excel VBA
- Data Processing

## Description

These tools are inspired by real-world telecom porting operations, including carrier interaction and order validation.

## Structure

python/ - Python scripts  
vba/ - Excel VBA automation  

## Future Improvements

- Convert to web application (React + Node.js)
- Add API integration
- Implement CI/CD

## VBA Automation

The VBA scripts are exported as .bas files for better code visibility and version control.

An Excel macro-enabled file (.xlsm) is included as a tool to run the automation.

Sample data is stored separately in .xlsx format to demonstrate clean data handling.
