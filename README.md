# Telecom Automation Tools

This repository contains automation tools for telecom number porting workflows.

## Features

- Python script to simulate pulling telephone number status from carriers
- Excel VBA automation for processing telecom order data
- Sample dataset for testing

## Technologies

- Python
- Excel VBA
- Data Processing

## Description

These tools are inspired by real-world telecom porting operations, including carrier interaction and order validation.

## Structure

python/ - Python scripts  
vba/ - Excel VBA automation  
sample_data/ - Sample data files
